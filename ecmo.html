How to order
-ecmo blood-


  • First open the patient's chart
  • Next select "Manage Orders."
  • On the far right of the screen select "Order-sets."
  • Then type "ECMO" into the search box.
  • Select "PEDS ECMO Prime Blood"
  • Select the blood bank order appropriate to the patients age/weight.
  • Select "STAT".
  • Select "Today".
  • Type the patient's blood band number.
  • Type the volume request to the appropriate tube station.
  • Select "Accept"
>
<

There is a checklist to ordering blood for ecmo in the ecmo binder in the monitor room.

If the patient does not have a blood band you are supposed to use the MRN number in place of the blood band number.

If Epic is down, and you need to order blood there in a ecmo blood order form as well as a request for service that needs to be completed and faxed down to blood bank.

Dont forget to call blood bank for all ecmo blood orders!
