General Review Icons


If you hover over a patient sector, the action buttons will appear. Hover again over review and you will have two options: alarm review & general review. Inside both these selections you will see many icons. Here is a review of what these icons do.



Delete Alarms:

When you select an alarm review the icons are in a column on the right. This particular icon deletes selected alarms and strips. Once deleted this action cannot be undone.


Save A Strip:

During your review you may need to save a strip. After you select the section of EKG that you want to save just click this icon. This is how you manually save strips throughout your review.


Filter Alarms:

You can filter the alarms, to easily view saved strips only or other alarms.


Print Strips:

You can print out strips with this button.


Record A Strip:

This icon allows you to record a strip. This will print a strip on telemetry paper.


Toggle View:

This can be used to toggle between different review windows, such as event view, tabular view and graphic view.


Large View:

Can be used to access a larger view of the review window section.


Edit Alarms:

This icon allows you to create, edit & delete custom alarm labels.


Navigation:

Used to navigate to the previous & next strips.


Settings:

Use to change the timeline or view duration. Changes you make to view or timeline duration only apply to the patient you are viewing. Once you close the window the changes revert back to the default settings.


Alarm Review:


To go through the alarms, first start by hovering over the patient sector. The action buttons will appear. Next hover over the review button and two options will appear. 'General Review' or 'Alarm Review'. Select alarm review.


The check boxes in the lower left allows the user to select alarms. The action tab on the right allows the user to perform several different tasks. You may recognize these icons as they are reviewed in detail above. These tasks include: toggling views, filtering alarms, printing strips, recording a strip, saving a strip & deleting strips.

Another option to be aware of is the select all feature which is not shown in this image. This option was a recent update and allows the user to select all the alarms at once.

To go through the alarms the user can 'select all' & then navigate through the alarms de-selecting 'true alarms'. Alternatively the user can also navigate through the alarms selecting only artifacts.

The alternative views to the alarm review bank are 'compressed view' & 'alarm window'.



Search Strips:


To find a strip easily you have a few options. First, if you saved the strip you can filter the alarms to show just the saved strips. You can also filter by yellow & red alarms. So if a patient alarmed for an event you can find it by filtering the type of alarms.

Another option is to 'search' for the strip. You can search by alarm name, or if you labeled your strip that you are searching for you can search using the label.

Using Calipers:


To use the calipers, check the box and slide the popup down so that its not in the way of your measurement.


View Vital Signs:


In the general review window select the toggle button, on the right side of the patient sector. Then you can adjust the timeline using the gears also on the right hand side. At the most you are able to adjust to a minute by minute view if needed.


Saving Strips:


There are two different types of save strips. First are strips saved of an arrythmia. Next are 'Control Strips'. Control strips are saved at certain times: 0600, 1400 & 2200. You can think of control strips like an experiment, the control strips are like the control group receiving a placebo. All saved strips are prepared in a report for cardiology to review.

To save an arrhythmia strip from general review, first click on the arrhythmia and line it up in the window. On the right side there is a little yellow comment box. Select this icon to save a strip, when saving the strip you can create a label, choose a generic label or leave the strip unlabeled.

During your review if you see the save strip icon at the bottom of the window it means a strip has been saved during that time frame.


Adjust Heart Rate Trend:


The image above is a trend of a patient's heart rate over a period of time. This is quite useful to quickly glance and view changes in heart rate which could signify an arrythmia.


Control Strips:

Control strips can be saved manually. This is done on each individual patient the same way you would save an arrhythmia strip. Or a less monotonous way would be selecting the save strip icon at the very top of each monitor. This will automatically save a strip for each patient on the monitor at the time that you save the strips. Just make sure to save at the correct times!
