View Patients on Server:


Start by rolling the mouse over a sector, the action buttons will appear at the bottom of the sector.

Click on patient window to use the drop-downs to select the unit and room number


ECG Analysis:


Start by hovering over the patient sector. The action buttons will appear. Next, hover over the button labeled 'patient window' a second option- ECG Analysis will dropdown. Select ECG Analysis.




In this window you can view all the leads, set primary & secondary leads, see beat annotation, relearn arrhythmias, and lastly change from multi-lead analysis to single-lead analysis.


Beat Annotation

Beat annotation is a great tool to see how the system is interpreting a rhythm. The letters above each beat allow you to see how the monitor is interpreting each beat. N = Normal, V = Ventricular.


RELEARN ARRHYTHMIA

The button labeled Relearn Arrhy can be used to relearn a rhythm when the monitor is misinterpreting an arrhythmia & causing artifact alarms


Multi-lead to Single-lead

Lastly in this window you can see the option to switch the mode from multi-lead analysis to single-lead analysis. This should be used only after other measures are taken such as, switching the primary & secondary leads, changing the electrodes and setup on the patient- (ensure the electrodes make good contact). Changing from multi to single-lead analysis is usually only necessary for some PACED patients that alarm consistently for artifacts. Alert the R.N caring for the patient if you believe this mode needs to be changed. Ensure the R.N is aware that by changing this mode the monitor will only analyze the primary lead.
